# Ej01Equipo

Es un CRUD de un equipo de futbol, en este caso el Atlético de Madrid.

## Principales tecnologías y herramientas utilizadas en el proyecto.

- Spring Boot
- Spring Data JPA
- MySQL 

## Características

(CRUD)

- Crear
- Leer
- Actualizar
- Eliminar 

## Uso

Mostrar equipo completo
http://localhost:8080/atleti
